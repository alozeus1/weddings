# Image Conventions

- `placeholders/`: temporary visuals for local development and tests.
- `couple/`: final couple portraits and hero images.
- `couple/optimized/`: web-sized shared derivatives used by hero banners, mosaics, and story timeline cards.
- `gallery/`: additional curated wedding photos shown on the gallery page.
- `gallery/optimized/`: resized gallery derivatives used by the live gallery grid instead of the original camera PNGs.
- `wedding-party/`: portraits for bridal party and groomsmen.
- `families/`: family portraits and heritage photos.
- `menu/`: menu photography and featured registry thumbnails.

Naming rules:
- lowercase, kebab-case.
- keep stable names referenced by content JSON files.
- replace file content in-place when switching from placeholders to final imagery.
